# Projet SeminaireMath

# Keybindings
	- Flêches de naviguation pour les translations
	- WASDQE pour les rotations
	- PO pour le scaling
	- L pour le shearing X-Y
	- K pour le shearing Y-Z
	- R reset le cube

# Problèmes connus
Les rotations ne s'ajoutent pas entre elles. Lors de la multiplication des quaternions, le résultat était incompréhensible, nous avons donc choisi de le retirer
